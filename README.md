

# Personal Portfolio Website

## Live Demo
[https://work-showcase-smhy.vercel.app/](https://work-showcase-smhy.vercel.app/)

## Overview
A responsive personal portfolio website built to showcase skills, projects, certifications, and contact information.

## Tech Stack
HTML  
CSS  
JavaScript  
Phosphor Icons  
Python (utility scripts)

## Project Structure
index.html - main page  
style.css - styling  
script.js - interactivity  
assets - images and certificates  
backend-js - utility scripts  
repos.json - cached data  

## Features
Modern responsive design  
Dark theme UI  
Project portfolio section  
Certifications gallery  
Contact section  
Smooth animations  

## Setup
Open index.html in browser  

OR run:
python -m http.server 8000  

## Scripts
fetch_repos.js - fetch GitHub repos  
update_cards.js - update project cards  
rewrite.js - generate light theme  
rewrite_dark.js - generate dark theme
